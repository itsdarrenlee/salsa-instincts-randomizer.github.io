# salsa-instincts-randomizer.github.io
Simple tool to randomize the pairwise combinations of Salsa combinations. All content here provided AS-IS and is the sole rights and property of Svetlana Ray's and her Salsa Instincts course. I do not own any content.

- `index.html` is the randomizer homepage.
- `challenge.html` is the 5-8 card Panda + Honey Badger challenge page.
- `figures.html` is the timestamp reference page.
- `_data/figures.yml` is the shared source of truth for the figure catalog.

For GitHub Pages, adding a new root-level `.html` or `.md` file creates a new page. Because this repo already has `_config.yml`, Jekyll data files like `_data/figures.yml` can be used directly from those pages.
